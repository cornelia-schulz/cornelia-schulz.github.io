JavaScript basics

How does JavaScript compare to HTML and CSS?

If you think of JavaScript in terms of our house analogy, whereby HTML are the foundations and walls and CSS is the design such as the colours of your walls, the size of your windows, etc, JavaScript makes things a bit more dynamic and interactive. For example if you press down the handle of your door, the door will open.

Use JavaScript to change the values and properties of any HTML tag on your website.

What are control flow and loops?

Control flow is the order in which the computer executes the code we write. The computer will also make decisions by executing code based on certain conditions. To take a real world example, we will look at making soup for dinner:

  1. Open fridge and check if there are all ingredients for my soup
  2. No? GO SHOPPING!!!
  3. Check fridge again. Are there now all ingredients I need?
  4. Yes? GET CHOPPING!!!
  5. Add the chopped ingredients one after the other into the pot
  6. Add water
  7. Stir!
  8. Add seasoning
  9. Bring to boil
  10. Stir!
  11. Keep stirring...
  12. Bon appetit!

And the code would probably look something like this:

What's the difference between accessing data from arrays and objects?

Objects in JavaScript are a collection of properties. To access information within an object you can use the dot or bracked notations. For example:

var dog = {
"heads": 1,
"tail": 1,
"legs": 4
}
To get the number of tails for your dog use:
dog.tail // 1
or
dog['tail'] // 1

Javascript arrays are a type of object used for storing multiple values in a single variable. Each value gets numeric index and may be any data type. They have a length property that you can use to see how many items are currently in the array. To access data within an array, use the index of the item you'd like to get to. Bear in mind that the index starts at 0. For example:

var myAnimals = ["dogs", "cats", "mice"];
 myAnimals[1]; // cats

The difference in access is that you use index numbers to access an array and a property name to access an object.

What are functions and why are they useful?

Functions are a fundamental building block in JavaScript. They form a set of procedures that the computer will run through when the function is called (invoked) for example by clicking on a button, etc.

function helloWorld(){
console.log("Hello World!");
}

This function will simply print the words "Hello World" to the console. Functions are extremely useful because they can be reused. You can call them again and again and don't have to repeat all the code that is contained in the function.
