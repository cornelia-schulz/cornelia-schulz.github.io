JavaScript Basics

How does JavaScript compare to HTML and CSS?

If you think of JavaScript in terms of our house analogy, whereby HTML are the foundations and walls and CSS is the design such as the colours of your walls, the size of your windows, etc, JavaScript makes things a bit more dynamic and interactive. For example if you press down the handle of your door, the door will open.

Use JavaScript to change the values and properties of any HTML tag on your website.

What are control flow and loops?

Control flow is the order in which the computer executes the code we write. The computer will also make decisions by executing code based on certain conditions. To take a real world example, we will look at making soup for dinner:

function goShopping(){
get up and go shopping;
return (full fridge);
}
function chop(ingredients){
get ingredients from fridge and cupboards and start chopping;
return (ingredients ready for cooking);
}
function boil(choppedIngredients){
for (each choppedIngredient){
add choppedIngredient to pot;
}
add water to the pot;
turn on the heat;
}
