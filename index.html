Cultural blogs

Reflections

Reflection is a mental process. It is contemplation or a long consideration. Thoughts or opinions that come to you while you are reflecting are called reflections. Unlike a reflection in a mirror, it in as interpretation of what is going on between learning and thinking.

Time and habits

Habits are rituals and behaviors that we perform automatically, allowing us to carry out essential activities such as brushing our teeth, etc. Our unconscious habits free up resources for our brains to carry out other more complex tasks like solving problems or deciding what to make for dinner.

Emotional intelligence

Emotional intelligence is being able to keep an eye on your own and others emotions in order to guide your thinking and your actions.

Technical blogs

HTML, CSS & DOM part 1

Styling HTML with CSS. CSS stands for Cascading Style Sheets. CSS describes how HTML elements are to be displayed on screen, paper, or in other media. CSS saves a lot of work. It can control the layout of multiple web pages all at once. CSS can be added to HTML elements in 3 ways: Inline - by using the style attribute or by using a separate css style sheet that is linked to your HTML.

HTML, CSS & DOM part 2

When a browser reads a style sheet, it will format the HTML document according to the information in the style sheet. Three Ways to insert CSS into your HTML. External style sheet; Internal style sheet; Inline style. External style sheet. All three will let you shape your HTML to make it look the way you want it to look.
